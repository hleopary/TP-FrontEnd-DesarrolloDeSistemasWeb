# Sprint 0 - Setup

## Objetivo

Dejar lista la base de trabajo para que el equipo pueda avanzar sin desorden.

## Tareas

- Crear repo publico.
- Confirmar estructura de carpetas.
- Definir naming de archivos.
- Preparar Obsidian con plantillas.
- Cargar plan de implementacion.

## Criterio de cierre

El sprint cierra cuando la estructura y la documentacion base estan listas para comenzar el frontend.

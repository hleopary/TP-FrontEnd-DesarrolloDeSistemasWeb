# Overview

## Objetivo

Documentar el alcance general del frontend, la propuesta elegida y la forma de trabajo del equipo.

## Resumen del proyecto

- Sitio frontal para el grupo Clouds.
- Estetica RPG con narrativa de gremio y combate contra el Bug.
- Una portada interactiva y una pagina personal por integrante.
- Bitacora y README obligatorios para cierre academico.

## Entregables

- Portada.
- Bitacora.
- 4 paginas individuales.
- Estilos globales.
- Script JS principal.
- Assets visuales.
- Documentacion final.

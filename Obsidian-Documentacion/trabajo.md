# Trabajo

Archivo de contexto para que las IAs del equipo trabajen con el mismo criterio.

## Convenciones

- Mantener nombres descriptivos.
- Evitar funciones duplicadas.
- Usar una sola estetica visual base.
- Documentar decisiones tecnicas y visuales.
- Priorizar soluciones simples y reutilizables.

## Antes de pedir codigo

- Revisar el plan de implementacion.
- Revisar las notas de sprint.
- Revisar el registro de decisiones.
- Revisar el contexto de la tarea concreta.

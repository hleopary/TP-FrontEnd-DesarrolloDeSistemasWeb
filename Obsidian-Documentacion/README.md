## Flujo de trabajo recomendado

1. Abrir INDEX.md.
2. Revisar Tareas.md y Kanban/Kanban-Tareas.md.
3. Planificar y ejecutar por sprint.
4. Registrar decisiones y uso de IA.
5. Registrar errores y soluciones.
6. Actualizar output con resultados relevantes.

## Regla de organizacion

- No crear notas sueltas en la raiz.
- Usar siempre la carpeta funcional correcta.
- Evitar duplicar informacion entre notas.
- Toda decision importante debe quedar registrada.

## Nota sobre desktop.ini

En Windows pueden regenerarse archivos desktop.ini automaticamente. No forman parte del contenido funcional del vault.
